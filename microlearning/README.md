# Financial literacy

  # Microlearning System

## Overview

The Microlearning System is the core educational component of the Financial Literacy Education App. It delivers personalized, bite-sized financial lessons that help users build financial knowledge one concept at a time instead of overwhelming them with long courses.

Each lesson is designed to take only a few minutes to complete, making financial education more accessible, engaging, and easier to retain.

---

## Purpose

Traditional financial education often presents too much information at once, leading to low engagement and poor retention.

The Microlearning System addresses this by:

- Breaking financial topics into small lessons
- Personalizing recommendations based on assessment results
- Tracking user progress
- Reinforcing learning through quizzes
- Encouraging consistent daily learning

---

## Core Features

### Personalized Learning Path

Users receive lessons based on:

- Assessment results
- Financial knowledge level
- Learning progress
- Previously completed lessons

---

### Bite-Sized Lessons

Each lesson includes:

- Learning objective
- Short explanation
- Real-world example
- Key takeaway
- Estimated completion time (3–5 minutes)

---

### Quiz Assessment

Every lesson concludes with a short quiz that:

- Measures comprehension
- Reinforces learning
- Unlocks future lessons
- Updates the learner's progress

---

### Progress Tracking

The system tracks:

- Lessons completed
- Quiz scores
- Completion percentage
- Learning streaks
- Topics mastered

---

### Recommendation Engine

Instead of recommending lessons randomly, the system uses deterministic logic to recommend the next lesson based on:

- User assessment
- Completed lessons
- Lesson prerequisites
- Difficulty level
- Current learning path

---

## Learning Categories

Example learning categories include:

- Budgeting
- Saving Money
- Credit Scores
- Debt Management
- Banking Basics
- Investing
- Insurance
- Taxes
- Retirement Planning
- Fraud Prevention

---

## User Flow

1. User completes financial assessment.
2. Assessment determines current knowledge level.
3. Personalized learning path is created.
4. User completes micro lesson.
5. User takes quiz.
6. Progress is recorded.
7. Recommendation engine selects the next lesson.
8. Repeat until learning path is complete.

---

## Data Model

The Microlearning System manages:

- Learning categories
- Lessons
- Quiz questions
- Quiz answers
- User progress
- Lesson prerequisites
- Learning paths
- Assessment results
- Recommendation logic
